# Expense Tracker App 💸

A mobile app to track income and expenses with real-time updates.

## Features

* Login & Register (Firebase Auth)
* Add Income & Expense
* Edit & Delete Transactions
* Modal-based actions
* Real-time Firestore database

## Tech Stack

* React Native (Expo)
* Firebase (Auth + Firestore)
* React Native Elements

## Screens

* Home
* Add Transaction
* All Transactions
* Update Transaction
